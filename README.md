# 📝 Task 1 - To-Do List

## 📌 Description
This project is a To-Do List application developed using Python. It helps users organize and manage their daily tasks through a simple and user-friendly interface.

## ✨ Features
- ➕ Add new tasks
- 📋 View all tasks
- ✏️ Update existing tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 🎨 Simple and interactive user interface

## 🛠️ Technologies Used
- Python

## 🚀 How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Run the Python file:
   ```bash
   python todo.py
   ```
   *(Replace `todo.py` with your actual file name if it is different.)*
